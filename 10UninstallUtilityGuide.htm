ArcGIS 10 Uninstall Utility for Windows 

This utility will assist you in locating and removing previously installed ArcGIS products on Windows only, in preparation for the installation of ArcGIS 10.

Overview

This utility will locate and automatically remove ArcGIS products previous to ArcGIS 10 that are installed on your machine. Uninstalling these products is required to install ArcGIS 10.

NOTE: All ArcGIS applications, processes, and ArcGIS Server services must be shutdown before using the Uninstall Utility. Applications, processes, and services include, but are not limited to, Server Object Manager (SOM), Server Object Container (SOC), ArcMap, ArcCatalog, and ArcReader. If any applications, processes, or services are running, the ArcGIS 10 Uninstall Utility may not successfully remove all products.

Using the ArcGIS 10 Uninstall Utility

  1. Launch the utility by clicking the ArcGIS 10 Uninstall Utility link or manually double-clicking 10UninstallUtility.exe.
  2. Conflicting products will be displayed.Click uninstall to remove all displayed products.
  3. After all displayed products have been removed, click exit to dismiss the utility.
  4. ArcGIS 10 can now be installed.

ArcGIS products this utility will uninstall

  • ArcGIS 10 Pre-Release - The ArcGIS 10 Uninstall Utility will remove all ArcGIS 10 Pre-Release software except ArcSDE and certain extensions.
  • ArcGIS 9.4 Beta 2 - The ArcGIS 10 Uninstall Utility will remove all ArcGIS 9.4 Beta 2 software except ArcSDE and certain extensions.
  • ArcGIS 9.4 Beta 1 - The ArcGIS 10 Uninstall Utility will remove all ArcGIS 9.4 Beta 1 software except ArcSDE and certain extensions.
  • ArcGIS 9.3.1 - The ArcGIS 10 Uninstall Utility will remove all ArcGIS 9.3.1 software except ArcSDE, ArcGIS License Manager and certain extensions. ArcGIS 9.3.1 Beta software will not be removed.
  • ArcGIS 9.3 - The ArcGIS 10 Uninstall Utility will remove all ArcGIS 9.3 software except ArcSDE, ArcGIS License Manager, and certain extensions. ArcGIS 9.3 Beta software will not be removed.
  • ArcGIS 9.2 - The ArcGIS 10 Uninstall Utility will remove all ArcGIS 9.2 software except ArcSDE, ArcGIS License Manager, and certain extensions. ArcGIS 9.2 Beta software will not be removed.
  • ArcGIS 9.1 - The ArcGIS 10 Uninstall Utility will remove all ArcGIS 9.1 software except ArcSDE, ArcGIS License Manager, and certain extensions. ArcGIS 9.1 Beta software will not be removed.
  • ArcGIS 9.0 - The ArcGIS 10 Uninstall Utility will remove all ArcGIS 9.0 software except ArcSDE, ArcGIS License Manager, and certain extensions. ArcGIS 9.0 Beta software will not be removed.

Notes

  • Product versions prior to 9.X are not detected. Manually uninstall these products (using the Operating System Add/Remove Programs).
  • The ArcGIS 10 Uninstall Utility will not detect third party extensions or plug-ins. Manually uninstall these products (using the Operating System Add/Remove Programs) before removing any ArcGIS products.
  • This utility is intended for Windows only.
